Operator MetricsTupleCounterHelper

Toolkits > com.ibm.streamsx.perf.measurements 1.0.0 > com.ibm.streamsx.perf.measurements > MetricsTupleCounterHelper

The MetricsTupleCounterHelper operator is to be used in conjunction with the MetricsTupleCounter operator. The MetricsTupleCounterHelper operator contains output functions for easier formatting of output.

The MetricsTupleCounterHelper and MetricsTupleCounter operator should be fused into the same Processing Element (PE).

Examples
See example in the MetricsTupleCounter documenation.

Parameters

This operator supports 1 parameter.
includeHeaders

If includeHeaders is set to true, attribute descriptions will be placed in an extra tuple prior to the first tuple being output. All attributes in the output stream must be of type rstring if includeHeaders is true. This value will default to false.

Properties
  • Type: boolean
  • Cardinality: 1
